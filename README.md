# Pixiv Ranking New Tab

## Install
https://chrome.google.com/webstore/detail/mjfpkofgblpbkmjapadadendndgpiehh

## Intro
Show Pixiv daily rankings on your new tab page.  
新しいタブページでPixivの毎日のランキングを表示します。  
在新标签页上显示Pixiv每日排名。  
在新標籤頁上顯示Pixiv每日排名。  

## Features

* Show Pixiv daily rankings
* Download illusts
